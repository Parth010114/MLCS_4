# MLCS_4
Pruning Defense for BadNets
Overview
This repository contains code for implementing and evaluating a pruning defense strategy applied to a BadNet to mitigate backdoor attacks. The defense involves selectively pruning channels from the last pooling layer based on average activation values.

Files and Directories
MLCS_Assignment4.ipynb: The main Jupyter Notebook containing the implementation of the pruning defense, loading data, and evaluating the model.
Dataset & Model - The dataset and badnet for this code have been uploaded on google drive and drive has been mounted in google colab. Before running the code please make sure to check if all the files are available
MLCS_merged.pdf - Report for this project. Contains the results that were found in this assignment.
